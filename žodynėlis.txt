Kompiliavimo žodynėlis
* Compile time - laikas kuomet programinis kodas paverčiamas kompiuteriui suprantamu mašininiu kodu.
* Programa - Sukompiliuotas(paruoštasas) mašininis kodas, kurį gali įvykdyti kompiuteris.
* Procesas - Šiuo metu vykdoma Programa.
* Runtime - Laikas, kurio metu vyksta procesas.
* Kompiliuoti - paversti aukštesnio lygio programavimo kalbos kodą į žemesnio lygio programavimo lygio kodą
* Transpiliuoti - paversti programavimo kalbos kodą į kitą, to paties lygio, programavimo kalbos kodą

Technologijų žodynėlis
* JavaScript - Prototipais grįsta objektiškai oprientuota kalba, kuri dažnai rašoma funkciniu būdu
* TypeScript - JavaScript kalbos papildymas sugriežtinant kodą tipais
* Node.js - C++ programa, skirta kompiliuoti ir vykdyti JavaScript kodą
* npm - komanda skirta tvarkyti bibliotekoms

Programavimo žodynėlis
* argumentas - reikšmė perduodama kviečiant funkciją
* parametras - funkcijos iškvietimo metu kuriamas kintamasis, kuris aprašomas funkcijos antraštėje
